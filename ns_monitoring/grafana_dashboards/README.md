# Grafana Dashboards

Aquesta carpeta conté dashboards industrials de Grafana utilitzats per monitorar el rusc intel·ligent en entorns de producció.

## Contingut

• Fitxers `.json` exportats de Grafana amb la configuració completa de panels.  
• Queries Flux utilitzades per consultar dades històriques des d’InfluxDB.  
• Configuracions de visualització per a:
  - acústica
  - sensors del rusc
  - meteorologia
  - estat energètic
  - alertes

## Notes

• Aquesta carpeta no forma part del dashboard Streamlit.  
• És un component extern del sistema, utilitzat en desplegaments industrials.  
• Els fitxers `.json` poden ser importats directament a Grafana.
